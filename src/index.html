Rest Api de bebidas

Enpoints

/bebidas/
Muestra una lista con todas las bebidas
/bebidas/
Inserta Una nueva bebida con los valores de los parametros
Parametros

Nombre: String

Descripcion: String

Precio: Number

Cantidad: Number

/bebidas/:id
Muestra muestra la bebida con el id especificado en la ruta
/bebidas/:id
Modifica la bebida con el id especificado en la ruta con los parametros
Parametros

Nombre: String

Descripcion: String

Precio: Number

Cantidad: Number

/bebidas/:id
Elminina la bebida con el id especificado en la ruta
